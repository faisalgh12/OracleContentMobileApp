# Assets Directory

This directory contains the app's visual assets.

## Required Files

### App Icons
- `icon.png` - Main app icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

### Splash Screen
- `splash.png` - App launch splash screen (1242x2436)

## Image Specifications

- **App Icon**: 1024x1024 PNG with transparent background
- **Adaptive Icon**: 1024x1024 PNG with safe area margins
- **Splash Screen**: 1242x2436 PNG optimized for mobile displays
- **Favicon**: 48x48 PNG for web browser tabs

## Design Guidelines

- Use Oracle brand colors (#FF0000, #000000)
- Ensure good contrast and readability
- Test on both light and dark backgrounds
- Optimize file sizes for mobile performance

## Placeholder Assets

For development, you can use placeholder images or create simple designs. The app will work without these files, but they're required for production builds.
